# AI Chatbot using Python

A simple AI chatbot built using Python and JSON-based intents. This chatbot responds to user messages based on predefined patterns and responses.

---

## Features

* Beginner-friendly Python project
* Uses JSON for chatbot intents
* Random chatbot responses
* Easy to customize and extend
* Console-based chatbot
* Lightweight and fast

---

## Technologies Used

* Python
* JSON
* Random Module

## How It Works

1. User enters a message.
2. The chatbot tokenizes the input.
3. It compares words with predefined intent patterns.
4. If a match is found, a random response is returned.
5. Otherwise, the chatbot replies with a default message.
